# Netflix Data Cleaning and Preprocessing

## Objective
The objective of this project was to clean and preprocess the Netflix Movies and TV Shows dataset by identifying and correcting common data quality issues using Microsoft Excel.

## Dataset
* **Dataset:** Netflix Movies and TV Shows
* **Source:** Kaggle

## Tools Used
* Microsoft Excel

## Data Cleaning Tasks Performed
* Identified and handled missing values by replacing blank cells with **"Unknown"** where appropriate.
* Checked for duplicate records using Excel's **Remove Duplicates** feature (no duplicate records were found).
* Verified text consistency across categorical columns.
* Corrected three misplaced duration values (`66 min`, `74 min`, and `84 min`) that were incorrectly stored in the **Rating** column by moving them to the **Duration** column.
* Verified column headers and ensured they were clean and consistent.
* Reviewed the dataset for data consistency and overall quality.

## Files Included
* `netflix_titles.csv` – Original dataset
* `cleaned_dataset.xlsx` – Cleaned dataset
* `README.md` – Project documentation

## Outcome
The dataset was successfully cleaned and prepared for further data analysis and visualization by handling missing values, verifying data consistency, and correcting data quality issues.
